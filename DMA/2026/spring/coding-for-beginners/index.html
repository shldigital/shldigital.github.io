Some useful terms:

Integrated Development Environment (IDE)
  • Programming languages 'interpret' plain text commands
  • IDEs are essentially text editors (like notepad)
  • They also package useful functionality that helps you edit and understand code, as well as code execution, building or visualisation tools

Python syntax for looping:


                print("Loop starting!")
                for number in range(10):
                    print("Loop number ", number)
                print("Loop finished!")

Note the indentation ‚á• before the print statement

This indentation is achieved using the tab key

The final line is not indented, and therefore is not a part of the loop

Try experimenting with indentation and with other parts of the code

A note on comments

"Comments" are text that you can insert into you program that are not processed by the interpreter. In python they start with the # character: 

# Here's where we start the loop:
for number in range(10): # BTW range(10) starts at 0!
    print("Loop number ", number)

üåü

A note on comments

Comments help you and others understand your code

This can be increasingly important for more complex code, but also code that you expect to maintain or change many years down the line

Python can also loop indefinitely with e.g. 

                while True:
                    print("I'm looping!")
Running an infinite loop (which you might do by accident as well as on purpose!) can be stopped with the STOP ‚õî button in thonny.
