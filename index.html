Python syntax for looping:


                print("Loop starting!")
                for i in range(10):
                    print("Loop number ", i)
                print("Loop finished!")

Note the indentation ‚á• before the print statement

This indentation is achieved using the tab key

The final line is not indented, and therefore is not a part of the loop

Try experimenting with indentation and with other parts of the code

Note there's an offset on the numbers that are output! How could we fix that?

Python can also loop indefinitely with e.g. 

                while True:
                    print("I'm looping!")
Running an infinite loop (which you might do by accident as well as on purpose!) can be stopped with the STOP ‚õî button in thonny.
