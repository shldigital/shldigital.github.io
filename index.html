Some useful terms:

Integrated Development Environment (IDE)
  • Programming languages 'interpret' plain text commands
  • IDEs are essentially text editors (like notepad)
  • They also package useful functionality that helps you edit and understand code, as well as code execution, building or visualisation tools

Some useful terms:

Integrated Development Environment (IDE)
  • Not the only way to interact with code
  • Can use regular text editors and the command line
  • Can use notebooks like google collab or jupyter
  • Can get AI assistants to write your code (vibe coding)
